# Image generation prompts

Use these prompts if replacing the current runtime-generated placeholder art with painted sprites.

## Style guide

Bright 2D cartoon browser game, late elementary school friendly, rounded shapes, thick outlines, colorful pretend science lab, whimsical and safe, no realistic hazardous chemicals, no scary injuries.

## Sprite prompts

- **Junior mad scientist:** cheerful child scientist with oversized goggles, purple gloves, curly hair, star stickers on lab coat, transparent background, 2D cartoon game sprite.
- **Reagent bottles:** six squat cartoon bottles labeled Fizz Powder, Sour Drops, Bubble Base, Purple Potion, Glimmer Salt, Goo Gel, distinct bright colors, thick outline, transparent background.
- **Flask reactions:** bubbling foam eruption, rainbow sludge swirl, glitter crystal jam, cork rocket, soot puff, slime monster, duck portal, separate transparent sprite sheets.
- **Lab background:** cozy classroom-safe fantasy lab bench, posters with science words, silly gadgets, no realistic hazard labels, wide 16:10 game background.
